The Hugging Face Datasets makes thousands of datasets available that can be found on the Hub. Check if there's any dataset you would like to try out!

In this tutorial, we will load the agnews dataset, a collection of more than 1 million news articles on four categories: world, sports, business, sci/tech.

2. Loading the dataset

The agnews dataset has data fields of:

  • text: a string feature.
  • label: a classification label, with possible values including World (0), Sports (1), Business (2), Sci/Tech (3).
from datasets import load_dataset
agnews = load_dataset('ag_news')
